# python-practice
Python  – Basic Programming Problems
📌 Description

This  contains basic Python programs designed to understand fundamental programming concepts such as:

User input
Data types
Type conversion
Arithmetic operations
Expressions
Swapping values
Mathematical formulas

🎯 Objectives
To learn how to take input from users
To perform basic arithmetic operations
To understand type casting (int, float, string)
To apply mathematical formulas in Python
To improve logical thinking and coding skills

🧾 List of Programs

✅ Q1: Name and Age
Takes user name and age and prints a formatted sentence.

✅ Q2: Arithmetic Operations
Takes two numbers and performs:
Sum
Difference
Product
Quotient

✅ Q3: Average Calculation
Takes two integers and one float, converts them to float, and calculates the average.

✅ Q4: Type Conversion
Converts a string number into:
Integer
Float
String again
Prints values along with their types.

✅ Q5: Expression Evaluation
Evaluates:
x = 10 + 3 * 2 ** 2
Demonstrates operator precedence:
Exponentiation (**)
Multiplication (*)
Addition (+)

✅ Q6: Swap Two Numbers
Swaps two numbers using a temporary variable.

✅ Q7: Celsius to Fahrenheit
Converts temperature using formula:
F = (C × 9/5) + 32

✅ Q8: Area of Circle
Calculates area using:
Area = π × r²   (π = 3.14)

✅ Q9: Simple Interest
Calculates SI using:
SI = (P × R × T) / 100

✅ Q10: Integer and Fractional Part
Separates:
Integer part
Fractional part
from a decimal number.

🛠️ Technologies Used
Python 3.x
VS Code / IDLE / PyCharm

📚 Concepts Covered
Variables and Data Types
Input/Output Functions
Type Casting
Arithmetic Operators
Expressions and Precedence
Basic Mathematics in Programming
